# Interactive Solar System Events

A futuristic, immersive 3D web application that lets you explore and interact with a solar system simulation.

## 🌌 Features

- **Full 3D Solar System**: All 8 planets orbiting the Sun with realistic proportions
- **Interactive Controls**: Drag to rotate camera, scroll to zoom, click on planets
- **"What If" Events**: Trigger cosmic events on each planet
  - **Earth**: Add a second moon, bring moon closer, add Saturn-like rings
  - **Saturn**: Drop in water (ripple effect), remove rings, double ring size
  - **Sun**: Solar flares, brightness boost, mini supernova
  - And many more for each planet!
- **Global Events**: Spawn a black hole that pulls nearby planets
- **Smooth Animations**: 60 FPS real-time rendering with particle effects
- **Futuristic UI**: Glassmorphism panels with neon glow accents

## 🛠️ Tech Stack

- **React** - UI framework
- **React Three Fiber** - 3D rendering in React
- **Three.js** - 3D graphics engine
- **Framer Motion** - UI animations
- **Tailwind CSS** - Styling

## 📦 Installation

```bash
npm install
```

## 🚀 Usage

```bash
npm run dev
```

Open your browser to `http://localhost:3000`

## 🎮 Controls

- **Left Click + Drag**: Rotate camera
- **Scroll**: Zoom in/out
- **Click on Planet**: Open event panel
- **Click Event Button**: Trigger cosmic event
- **Close Panel**: Click ✕ in top right

## 🌟 Planet Events

### ☀️ Sun
- Solar Flare: Trigger a massive solar flare burst
- Super Brightness: Increase sun intensity dramatically
- Mini Supernova: Simulate a mini supernova explosion

### 🌍 Earth
- Second Moon: Add a second moon orbiting Earth
- Bigger Moon: Bring the moon closer and make it larger
- Saturn Rings: Add Saturn-like rings to Earth

### 🪐 Saturn
- Drop in Water: Show ripple simulation effect
- Remove Rings: Strip Saturn of its rings
- Double Rings: Make the rings twice as large

### And many more for Mercury, Venus, Mars, Jupiter, Uranus, and Neptune!

## 🎯 Experience

This is not just a visualization—it's an interactive space simulation where you can experiment with cosmic events in real-time. Feel like a cosmic architect as you reshape the solar system!
